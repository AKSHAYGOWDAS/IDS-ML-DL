# IDS-ML-DL

M.Tech thesis project on **Deep Learning based Intrusion Detection System (IDS)**.

## 📌 Project Overview
This project focuses on detecting network intrusions using deep learning and machine learning models.  
Experiments are conducted on **IoT and VeReMi datasets** to identify malicious traffic accurately.

## 🧠 Models Used
- Convolutional Neural Networks (CNN)
- XGBoost (for hybrid intrusion detection)
- Trained deep learning models (`.h5`, `.json`)

## 🚀 How to Run

1. Install dependencies:
pip install -r requirements.txt
2. Run the application:
python app.py

## 📊 Results
The proposed IDS achieves high detection accuracy on both **IoT** and **VeReMi** datasets.  
Result images are included in the repository.
